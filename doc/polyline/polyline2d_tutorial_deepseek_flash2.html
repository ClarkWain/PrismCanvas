1. 什么是 Polyline2D？

Polyline2D 是一个 轻量级 C++ 库，它的核心作用只有一句话：

给定一组 2D 点和一个线宽，生成一个三角形网格，渲染出一条有厚度的折线。

为什么需要它？因为 OpenGL 等图形 API 原生只支持 1 像素宽的线。要画出粗细可控、边缘圆滑的线条，必须自己把线条"展开"成三角形。

📌
输入：点序列
📐
输出：三角形网格
🎨
支持：3种连接样式 + 4种端点样式
听起来很复杂？别担心——我们会从最基础的 2D 向量开始，一步一步走到完整的 Polyline2D 实现。每个概念都有交互式示例。

2. 预备知识：2D 向量

在理解 Polyline2D 之前，我们必须先熟悉 2D 向量。一个向量 v 由两个分量 (x, y) 组成：

v = (x, y)  ,  |v| = √(x² + y²)

向量加法

a + b = (ax+bx, ay+by)

向量减法

a − b = (ax−bx, ay−by)

标量乘法

a × t = (ax·t, ay·t)

点积 (Dot Product)

a · b = axbx + ayby

叉积 (Cross Product, 2D)

a × b = axby − aybx

二维叉积的结果是一个 标量（在 2D 中，它代表两个向量围成的平行四边形的"有向面积"），它的符号告诉我们转向方向：

a × b > 0  ⟹  b 在 a 的逆时针方向
a × b < 0  ⟹  b 在 a 的顺时针方向

拖拽箭头末端调整向量。观察它们的和、差、点积和叉积如何变化。

3. 点、线段与方向向量

一条线段有两个端点 P0 和 P1。它的方向向量是：

d = P1 − P0 = (x1−x0, y1−y0)

我们通常把方向向量归一化（长度变为 1），方便后续计算：

d̂ = d / |d| = (dx, dy) / √(dx² + dy²)

线段的参数方程可以描述线段上的任意点：

P(t) = P0 + d̂ · t  ，其中  t ∈ [0, L] ，L = |d|

线段上的点随 t 在 0 ~ L 之间变化。观察方向向量和归一化方向向量。

4. 法向量：给线条"厚度"的关键

法向量（Normal）是与线段垂直的向量。对于一个方向向量 d = (dx, dy)，它的法向量是：

n = (−dy, dx)   (逆时针旋转 90°)
⟂
法向量与方向向量垂直
1
归一化后的法向量长度为 1
2
一个线段有两个法向量（±n）

代码实现非常简单（来自 LineSegment.h）：

struct LineSegment {
    Vec2 a, b;

    // 返回线段的法向量（逆时针旋转 90°）
    Vec2 normal() const {
        auto dir = direction();     // 归一化方向向量
        return {-dir.y, dir.x};    // (−dy, dx)
    }
};

绿色箭头 = 方向向量，蓝色箭头 = 法向量（两个方向都显示）。确认法向量与方向向量垂直。

为什么法向量重要？给一条细线加"厚度"，本质上是把线段沿法向量方向"推开"：两个边 = 中心线 ± 法向量 × 半厚度。这是整个 Polyline2D 的几何基础！

5. 偏移：从细线到粗线

现在，让我们把一条细线变成一条粗线。给定中心线段 C 和厚度 w：

半厚度 = w / 2
边1 = C + n̂ × (w/2)   （沿法向量偏移）
边2 = C − n̂ × (w/2)   （沿负法向量偏移）

这正是 PolySegment 的构造逻辑（来自 Polyline2D.h）：

template<typename Vec2>
struct PolySegment {
    LineSegment<Vec2> center, edge1, edge2;

    PolySegment(const LineSegment<Vec2> &center, float thickness)
        : center(center)
        // center.normal() × thickness  → 沿法向量偏移半厚度
        , edge1(center + multiply(center.normal(), thickness))
        , edge2(center - multiply(center.normal(), thickness)) {}
};

有了 edge1 和 edge2，我们可以用两个三角形（形成一个四边形）来渲染这一个线段：

三角形 1：(edge1.a, edge2.a, edge1.b)
三角形 2：(edge1.b, edge2.a, edge2.b)

Polyline2D 的 create() 方法中正是这样输出的：

// 为每个线段输出 6 个顶点（两个三角形）
*vertices++ = start1;    // edge1.a
*vertices++ = start2;    // edge2.a
*vertices++ = end1;      // edge1.b
*vertices++ = end1;      // edge1.b
*vertices++ = start2;    // edge2.a
*vertices++ = end2;      // edge2.b

调整厚度滑块，观察中心线如何被"展开"成两个边，再被两个三角形覆盖。

6. 线-线求交：尖角的秘密

当两段粗线段在拐角处相遇时，如果不做任何处理，会出现一个缺口。要填平这个缺口，我们需要计算两条边线的交点。

两条无限直线的交点公式：给定线段 A (A0→A1) 和 B (B0→B1)，参数化表示为：

A(t) = A0 + r·t ， B(u) = B0 + s·u
其中 r = A1−A0，s = B1−B0

求解 t 和 u：

t = (q × s) / (r × s)  ，u = (q × r) / (r × s)
其中 q = B0 − A0

如果分母 r × s = 0，则两条线平行（无交点）。如果 t ∈ [0,1] 且 u ∈ [0,1]，交点在线段范围内。

// 来自 LineSegment.h
static auto intersection(const LineSegment &a, const LineSegment &b,
                         bool infiniteLines) {
    auto r = a.direction(false);   // A1 - A0
    auto s = b.direction(false);   // B1 - B0
    auto q = subtract(b.a, a.a);   // B0 - A0

    auto denom = cross(r, s);
    if (abs(denom) < 0.0001f) return nullopt; // 平行

    auto t = cross(q, s) / denom;
    auto u = cross(q, r) / denom;

    if (!infiniteLines && (t < 0 || t > 1 || u < 0 || u > 1))
        return nullopt;

    return add(a.a, multiply(r, t));
}

7. 连接点样式：Miter / Bevel / Round

这是 Polyline2D 的核心功能之一。当折线拐弯时，需要决定如何填充拐角处的"缺口"。有三种选择：

MITER
尖角连接：两边外边缘延伸相交
BEVEL
平角连接：用三角形切掉尖角
ROUND
圆角连接：用多个三角形模拟圆弧

Miter（斜接）

Miter 连接通过计算两条边线的交点来延伸拐角。但有一个Miter 限制：如果夹角太小（< 20°），斜接会伸得太远，此时自动降级为 Bevel。

innerAngle < 20°  ⟹  自动切换为 BEVEL 
// 来自 Polyline2D.h
if (jointStyle == JointStyle::MITER
    && wrappedAngle < miterMinAngle) {
    jointStyle = JointStyle::BEVEL;  // 降级！
}

Bevel（斜切）

Bevel 连接识别出内边和外边，计算内边交点，然后用一个三角形填充外边与内边之间的区域。

顺时针判断：dir1 × dir2 < 0 → 顺时针
顺时针时：外边 = edge1, 内边 = edge2
逆时针时：外边 = edge2, 内边 = edge1

Round（圆角）

与 Bevel 类似，但用三角形扇（多个三角形）代替单个三角形来模拟圆弧。每个三角形覆盖约 10° 的弧。

三角形数量 = ⌊关节角度 / 10°⌋

三种连接样式对比。拖拽中间的点改变拐角角度，观察不同样式的表现。

Polyline2D 的 JointStyle 枚举：
MITER — 尖角（夹角太小自动降级为 BEVEL）
BEVEL — 平角
ROUND — 圆角

8. 端点样式：Butt / Square / Round / Joint

折线的起点和终点也需要特殊处理。Polyline2D 支持四种端点样式：

BUTT
平头：在端点处截断
SQUARE
方头：向外延伸半厚度
ROUND
圆头：用三角形扇画半圆
JOINT
闭合：首尾相连（用于多边形）

BUTT（平头）

最简单，直接使用边线的端点作为网格的端点，不加任何额外几何体。

SQUARE（方头）

将端点沿方向向量再延伸半厚度：

pathEnd = pathEnd + d̂ × (thickness / 2) 
if (endCapStyle == EndCapStyle::SQUARE) {
    pathEnd1 = add(pathEnd1, multiply(
        firstSegment.edge1.direction(), thickness));
}

ROUND（圆头）

在起点和终点各画一个半圆（三角形扇），半径 = 厚度 / 2。

JOINT（闭合）

在首尾之间创建一个额外的连接段，然后按 JointStyle 处理这个连接点。

四种端点样式对比。JOINT 模式会闭合路径。

9. 三角形扇：绘制圆角

三角形扇（Triangle Fan）是一种用一组三角形"铺满"一个扇形区域的技术。Polyline2D 用它在 Round 关节和 Round 端点中画圆弧。

核心思路：

1. 计算起始点角度：θ1 = atan2(y1, x1)
2. 计算结束点角度：θ2 = atan2(y2, x2)
3. 计算跨过的角度差：Δθ = θ2 − θ1
4. 用 N 个三角形均匀覆盖 Δθ，每个约 10°

2D 旋转公式（将点围绕原点旋转 rot 弧度）：

x' = cos(rot) · x − sin(rot) · y
y' = sin(rot) · x + cos(rot) · y 
// 来自 Polyline2D.h 的 createTriangleFan()
for (int t = 0; t < numTriangles; t++) {
    float rot = (t + 1) * triAngle;
    endPoint.x = cos(rot) * point1.x - sin(rot) * point1.y;
    endPoint.y = sin(rot) * point1.x + cos(rot) * point1.y;

    // 输出三角形 (startPoint, endPoint, connectTo)
    *vertices++ = startPoint;
    *vertices++ = endPoint;
    *vertices++ = connectTo;

    startPoint = endPoint;
}

10. 三角形网格：Polyline2D 的输出

现在让我们把所有的拼图合在一起，看看 Polyline2D 的完整算法流程：

Step 1

输入点 [P0, P1, ..., Pn-1]，过滤重复点

Step 2

对每对相邻点创建 PolySegment（中心线、边1、边2）

Step 3

处理端点样式（BUTT/SQUARE/ROUND/JOINT）

Step 4

遍历所有段，每段调用 createJoint() 处理连接处

Step 5

每段输出 6 个顶点（2 个三角形）

Step 6

返回完整的三角形顶点数组
create() 的完整流程（伪代码）：

1. thickness /= 2
2. 对每对点 (i, i+1)：
   如果点不同：segments += PolySegment(Line(pi, pi+1), thickness)
3. 如果是 JOINT 模式：添加首尾连接段
4. 根据 EndCapStyle 处理端点
5. 对每个 segment[i]：
   调用 createJoint(segment[i], segment[i+1], jointStyle)
   输出 6 个顶点
6. 返回顶点迭代器

third_party/polyline2d/include/LineSegment.h — 线段与求交
